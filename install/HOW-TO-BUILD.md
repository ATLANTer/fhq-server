# HOW-TO-BUILD

## Requirements Qt

```
$ sudo apt install g++ make qtchooser qt5-default libqt5websockets5 libqt5websockets5-dev libqt5sql5-mysql
```

## New Requirements

```
$ sudo apt install libwebsockets-dev libcurl4-openssl-dev
```

## Building
```
$ cd ..
$ qmake && make
```

## Install 
```
$ make && sudo make install
```
