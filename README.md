	* [SYSTEMD](install/SYSTEMD.md)

## (Ubuntu 16.04) Install from ppa:

```
$ sudo add-apt-repository ppa:freehackquest/fhq-server
$ sudo apt update
$ sudo apt install fhq-server
```

After install please try [CONFIGURING](install/CONFIGURING.md)
